# 智能体市场

智能体市场是Nexent平台即将推出的功能模块，将为您提供丰富的智能体资源。

## 🎯 功能预告

智能体市场将提供以下功能：

- **浏览智能体**：浏览平台提供的各类智能体
- **获取智能体**：一键获取并使用优质智能体
- **分享智能体**：分享您开发的智能体到市场
- **评价和反馈**：对智能体进行评价和反馈
