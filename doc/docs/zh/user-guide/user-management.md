# 用户管理

用户管理是Nexent平台即将推出的功能模块，将为您提供完整的用户管理能力。

## 🎯 功能预告

用户管理将提供以下功能：

- **用户列表**：查看和管理所有系统用户
- **用户权限**：配置用户的访问权限和功能权限
- **用户角色**：管理用户角色和权限组
- **用户统计**：查看用户使用情况和统计数据
